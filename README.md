# IRTH – Smart Heritage Tourism Mobile Application

## Overview
IRTH is a smart mobile application designed to enhance the cultural and historical tourism experience in Saudi Arabia. The application provides visitors with a centralized platform to discover heritage sites, plan trips, receive personalized recommendations, and report issues directly to tourism authorities.

This project was developed as part of the **IT423 – Introduction to IT Project Management** course at King Saud University.

## Features
- User registration and authentication
- Personalized trip planner
- Smart heritage site recommendations
- Interactive maps with GPS navigation
- Search and filter historical destinations
- Issue reporting system
- Bilingual support (Arabic & English)
